# ADR-011: Acordo ≠ Novação

- **Status:** Aceita
- **Data:** 2026-06 (reforçada 2026-07-13)
- **Decisores:** Luís (técnico) + Vicente/Cláudio (negócio), conforme registro no doc 02/CLAUDE.md

**Contexto.** O termo "renegociação" mistura dois mecanismos juridicamente distintos.

**Decisão.** Acordo (recuperação branda): dilui parcelas em atraso SEM liquidar o contrato — parcelas cobertas recebem vínculo de acordo. Novação (radical): liquida o contrato inteiro (LIQUIDADO_POR_NOVACAO) e cria um novo. Nunca confundir; o acordo é da CONTA (ADR-006).

**Consequências.** Trilhas, telas e contabilidade separadas para cada mecanismo.
